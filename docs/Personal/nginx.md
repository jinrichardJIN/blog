# nginx

#### 1.查看nginx安装目录

- whereis nginx
- rpm -ql nginx  (需要已经安装yum)

#### 2.nginx重启

- nginx 启动
- nginx -s reload (重新加载)
- nginx -s stop (强制退出)
- nginx -s quit (温和退出)
- systemctl 系列

#### 3.nginx 查看进程

- ps aux | grep nginx (查看哪些nginx正在运行)
